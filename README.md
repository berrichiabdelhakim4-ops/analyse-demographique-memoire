# analyse-demographique-memoire
Codes sources, tests de stationnarité (ADF) et projections démographiques à l'horizon 2050.
# Analyse et Projections Démographiques

Ce dépôt rassemble les scripts et livrables informatiques développés dans le cadre de mon mémoire de fin d'études.

## 📁 Structure du Projet
* analyse_demographique.py : Script Python principal (importation des données, modélisation avec Pandas, tests de racine unitaire Dickey-Fuller augmenté).
* /visualisations : Graphiques générés (Évolution des naissances 1960-2020, Taux de mortalité, Projections de la population).

## 🛠️ Spécifications techniques
* Langage : Python 3
* Librairies : Pandas, NumPy, Statsmodels, Matplotlib, Seaborn.
